# IARE Event App

The Official App for IARE to hold virtual events.

# Features
Judge and Participant Login(Admin/user) \n
Feature for participant to upload his/her work(picture/video) \n
Feature for Admin to view participants work of any specific contest \n

# Tools Used
Flutter
Firebase
